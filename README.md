# Yampi-PHP-SDK

SDK oficial da plataforma de e-commerce [Yampi](https://yampi.com.br).

## Principais Recursos

* [x] Recurso de Login com JWT e Tokens de usuário.
* [x] Recurso de Requests.

## Dependências

* PHP >= 8.2

## Instalação via Composer

```bash
$ composer require somosyampi/yampi-php-sdk
```
